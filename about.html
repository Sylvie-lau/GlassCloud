About This Project

• GlassCloud is a solo e-commerce project I conceived, designed, and developed from scratch.

• This project demonstrates the complete workflow of turning a creative idea into a functional online store using modern technologies.

• The product visuals are not traditional photography. I crafted them by writing precise descriptive text based on my aesthetic preferences for glass art, then iteratively refining those descriptions with an AI to generate the final, stylized images.

Vision

  • GlassCloud is a complete brand and e-commerce platform, conceived, designed, and developed independently from scratch.
  • This project aims to practice a modern, agile development workflow: rapidly transforming a creative vision into a market-ready product.
  • While not perfect, especially on the backend, I did my best to build out all necessary features, creating a complete process that combines cutting-edge AI tools with solid web development techniques.

Execution

  • In this project, I took end-to-end ownership of the entire lifecycle, from high-level brand strategy and UI/UX design to the specifics of front-end and back-end development.
  • My role was a blend of technical implementation and creative conception, ensuring a seamless synergy between the site's visual aesthetics and technical functionality.
  • A key innovation in this project's workflow is that all visual assets are driven by AI generative tools. My expertise is not in 3D modeling, but in directing these advanced tools to produce high-quality, on-brand visuals with exceptional speed and efficiency.

Key Competencies

Complete Project Construction

Demonstrated ability to independently build a web application from the front-end user interface to back-end data management, taking it from concept to a deployed product.

UI/UX & Interface Design

Focused on translating a brand concept into a fluid, intuitive, and visually appealing user interface. Ensured a consistent and enjoyable experience on any device through thoughtful animations and responsive design.

Front-End Dynamic Interaction

Utilized various front-end technologies to build a modern, high-performance interactive experience.

  • Animation: Used GSAP and ScrollTrigger for complex, scroll-based animations.
  • Componentization: Integrated React on the homepage to build a reusable, dynamic bestsellers carousel.
  • Visual Effects: Implemented Splitting.js for character-level text animations and AOS for module entrance effects.
  • Responsive Design: Ensured a flawless presentation on all devices using modern CSS layouts.

Backend Services & Database

Used Supabase as an all-in-one backend solution to provide robust and stable data support for the website.

  • Database: Leveraged Supabase DB (PostgreSQL) to store and manage all product, order, and user data.
  • Authentication: Integrated Supabase Auth for a complete user system, including email/password login and Google OAuth.
  • Serverless Functions: Wrote and deployed Supabase Edge Functions to securely handle backend logic, like creating Stripe payment intents.

Third-Party Service Integration

Successfully integrated the Stripe payment gateway to handle secure online credit card transactions. Also implemented Google Login, providing users with a convenient third-party authentication option to enhance the sign-up and login experience.

AI-Assisted Creative Design

Leveraged generative AI tools to create the visual content required for the project.
